### **Intro**

Divides damage dealt by a value of choice to free up some screen space. This is active only past level 64.

Fixed version of: https://github.com/Leyki/short-dmg-numbers

#### Settings

- Edit `settings.json`:
  - `enabled` - enables/disables the module.
  - `pvp_divisor` - pvp divisor to be used - 1 by default.
  - `pve_divisor` - pve divisor to be used - 1000 by default.

#### In-game Commands

- **_`smn`_** _`[on/off]`_ - turns on and off the module.
- **_`smn`_** _`pve`_ _`divisor(number)`_ - sets a new pve divisor.
- **_`smn`_** _`pvp`_ _`divisor(number)`_ - sets a new pvp divisor.
